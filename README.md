# Retail Sales Analysis Dashboard

## Project Overview
This project analyzes retail sales data using Power BI to identify sales trends, supplier performance, and warehouse vs retail sales.

## Tools Used
- Power BI
- Microsoft Excel

## Files Included
- Dataset.xlsx
- cleaned_data.xlsx
- PYTHON PROJECT.pbix
- Dashboard.jpg

## Features
- Sales performance analysis
- Monthly sales trends
- Item-wise sales analysis
- Top suppliers analysis
- Retail vs Warehouse comparison
- Interactive dashboard
